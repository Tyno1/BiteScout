# TextArea

A multi-line text input component that supports various configurations and themes. The TextArea component follows the same design system as other form components with consistent styling and accessibility features.

## Usage

Import the TextArea component from your atoms folder. Use it to create multi-line text inputs with labels, icons, and validation states. The component supports different themes, outline types, and sizes that match your design system.

## Props

### Required Props

- **label** (string) - The label text for the textarea
- **name** (string) - The name attribute for the textarea

### Display Props

- **useLabel** (boolean, default: false) - Whether to display the label
- **outlineType** (round, bottom, none) - The outline style of the textarea
- **icon** (ReactNode) - Optional icon to display on the left
- **rightButton** (ReactNode) - Optional button to display on the right
- **fullWidth** (boolean, default: false) - Whether the textarea should take full width
- **labelRow** (boolean, default: false) - Whether to display label in a row layout

### Behavior Props

- **required** (boolean, default: false) - Whether the textarea is required
- **onChange** (function) - Change event handler
- **value** (string) - The textarea value
- **placeholder** (string) - Placeholder text
- **rows** (number, default: 4) - Number of visible rows

### Styling Props

- **theme** (light, dark, transparent, default: light) - The theme of the textarea
- **inputSize** (sm, md, lg, default: sm) - The size of the textarea
- **iconStyle** (string, default: "") - Custom styles for the icon
- **rightButtonStyle** (string, default: "") - Custom styles for the right button
- **labelStyle** (string, default: "") - Custom styles for the label
- **textareaClassName** (string) - Custom styles for the textarea element

### Validation Props

- **errorMessage** (string) - Error message to display
- **helperText** (string) - Helper text to display below textarea

### Optional Props

- **id** (string) - Optional custom ID for the textarea
- **rightButtonOnClick** (function) - Click handler for the right button
