# Select

A dropdown select component that supports options, themes, and various configurations. The Select component follows the same design system as other form components with consistent styling and accessibility features.

## Usage

Import the Select component from your atoms folder. Use it to create dropdown selections with options, icons, and validation states. The component supports different themes, outline types, and sizes that match your design system.

## Props

### Required Props

- **label** (string) - The label text for the select
- **name** (string) - The name attribute for the select
- **options** (Option[]) - Array of options with value and label properties

### Display Props

- **useLabel** (boolean, default: false) - Whether to display the label
- **outlineType** (round, bottom, none) - The outline style of the select
- **icon** (ReactNode) - Optional icon to display on the left
- **rightButton** (ReactNode) - Optional button to display on the right
- **fullWidth** (boolean, default: false) - Whether the select should take full width
- **labelRow** (boolean, default: false) - Whether to display label in a row layout

### Behavior Props

- **required** (boolean, default: false) - Whether the select is required
- **onChange** (function) - Change event handler
- **value** (string) - The selected value
- **placeholder** (string) - Placeholder text for the first option

### Styling Props

- **theme** (light, dark, transparent, default: light) - The theme of the select
- **inputSize** (sm, md, lg, default: sm) - The size of the select
- **iconStyle** (string, default: "") - Custom styles for the icon
- **rightButtonStyle** (string, default: "") - Custom styles for the right button
- **labelStyle** (string, default: "") - Custom styles for the label
- **selectClassName** (string) - Custom styles for the select element

### Validation Props

- **errorMessage** (string, ReactNode) - Error message to display
- **helperText** (string) - Helper text to display below select

### Optional Props

- **id** (string) - Optional custom ID for the select
- **rightButtonOnClick** (function) - Click handler for the right button
