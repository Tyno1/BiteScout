# Spinner

A simple loading spinner component that provides visual feedback during asynchronous operations. The Spinner component uses a consistent design with smooth animations and proper accessibility features.

## Usage

Import the Spinner component from your atoms folder. Use it to indicate loading states in your application. The component provides a clean, centered loading indicator that fits seamlessly with your design system.

## Props

Currently, the Spinner component is a simple implementation with no configurable props. It provides a consistent loading experience across your application.

## Accessibility

- The spinner uses semantic HTML and proper ARIA attributes
- Provides visual feedback for screen readers
- Maintains focus management during loading states
